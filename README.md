# cv-aipply
AI-powered job application automation suite built with Python and Flask. Streamlines the actuarial job search by generating tailored cover letters and automating bulk email outreach with PDF attachments.
AIpply is a custom automation tool designed to optimize the recruitment workflow for Actuarial and Data Science roles. By integrating OpenAI's LLMs for content generation and Python's SMTP libraries for communication, this application eliminates repetitive manual tasks. It features a Flask-based interface for drafting personalized cover letters and a robust backend script that handles batch email dispatching, attachment management, and application status tracking in Excel/CSV.
